# Progress Block

Прототип блока **Progress** для мобильных веб-приложений.  
Блок предназначен для отображения процесса выполнения задач и их прогресса.

## Описание

Проект реализует блок прогресса с несколькими состояниями и API для управления.  
Основные возможности:

- Настраиваемое значение прогресса (`Value` от 0 до 100)
- Анимация вращения блока (`Animate`)
- Скрытие блока со страницы (`Hide`)
- Лёгкая переиспользуемость в других приложениях
- Адаптивная верстка для разных ориентаций экрана

## Состояния блока

1. **Normal** – базовое состояние, в котором можно управлять размером дуги через параметр `Value`.  
   - Дуга начинается с «12 часов» и двигается по часовой стрелке.
   - При `Value = 100` дуга возвращается в начальное положение.

2. **Animated** – блок вращается с заданным периодом по часовой стрелке.

3. **Hidden** – блок скрыт со страницы.

## API

Блок Progress предоставляет следующие возможности для управления через JavaScript:

- `setValue(value: number)` – устанавливает значение прогресса от 0 до 100.
- `setAnimated(on: boolean)` – включает или выключает анимацию блока.
- `setHidden()` – скрывает или отображает блок на странице.

## Использование

В приложении есть элементы управления:

- **Value** – текстовый ввод числа от 0 до 100
- **Animate** – переключатель включения/выключения анимации
- **Hide** – переключатель скрытия блока
